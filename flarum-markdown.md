# Markdown 插件

[![GitHub license](https://img.shields.io/github/license/flarum/markdown?color=blue)](https://github.com/flarum/markdown/blob/master/LICENSE) [![Latest Stable Version](https://img.shields.io/packagist/v/flarum/markdown.svg)](https://packagist.org/packages/flarum/markdown) [![Downloads](https://img.shields.io/packagist/dt/flarum/markdown.svg)](https://packagist.org/packages/flarum/markdown)

Flarum 原生插件，在安装 Flarum 时已经附带，无需手动安装。

文本编辑器加入 Markdown 语法支持。

### 重新安装

```
composer require flarum/markdown
```
