# Emoji 表情插件

Flarum 原生 Emoji 表情插件，在安装 Flarum 时已经附带，无需手动安装。

开启后，可以通过在文本编辑器中键入英文冒号 `:` 或点击图标激活表情插入。

### 重新安装

```
composer require flarum/emoji
```
